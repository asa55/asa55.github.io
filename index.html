For those of you who dare to follow this link, you'll find my Matlab code hosted in GitHub, and the ReadMe.md will tell you how to use it and links you to a report I wrote on neural networks that dives deep into the inner workings of this code.

Pi Falls from the Universe Yet Again

There is an awesome phenomenon in physics that enables you to calculate digits of pi via elastic collisions. This simple simulation verifies this to be true, and my friend and I derived this analytically as well. Detailing our proof in this Notebook is an open TODO item, but there are existing proofs we point you to in the meantime. To get started with this, click my link then when the next page fully loads click the "index.ipynb" file. This will let you experiment with the Python code live.
