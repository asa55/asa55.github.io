How not to implement Neural Nets

This code is something I did for an ML class. It implements a single hidden layer NN for character recognition from fundamental principles and works great, but unless you love math (I do) your might prefer to just start learning Keras with any backend (TF/Theano/CNTK). For those of you who dare to follow this link, you'll find my Matlab code hosted in GitHub, and the ReadMe.md will tell you how to use it and links you to a report I wrote on neural networks that dives deep into the inner workings of this code.

The Most Basic HTML/CSS/JS App: Part I (for web)

This app was the result of a suggestion from a friend. I am interested in exploring how to deploy web apps as native desktop apps using Electron. Pi Falls from the Universe Yet Again

There is an awesome phenomenon in physics that enables you to calculate digits of pi via elastic collisions. This is inspired by "The Most Unexpected Answer to a Counting Puzzle" on YouTube by 3Blue1Brown, and thoroughly explores the phase space representation of the problem. QuickStart guide: click my link then when the next page fully loads click the "index.ipynb" file. You can explore the visualizations and simulations I wrote in Python and even experiment with the Python code directly (via your browser).
