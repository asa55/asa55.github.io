How not to implement Neural Nets

This code is something I did for an ML class. It implements a single hidden layer NN for character recognition from fundamental principles and works great, but unless you love math (I do) your might prefer to just start learning Keras with any backend (TF/Theano/CNTK). For those of you who dare to follow this link, you'll find my Matlab code hosted in GitHub, and the ReadMe.md will tell you how to use it and links you to a report I wrote on neural networks that dives deep into the inner workings of this code.

The Most Basic HTML/CSS/JS App: Part I (for web)

This app was the result of a suggestion from a friend. I am interested in exploring how to deploy web apps as native desktop apps using Electron. My friend suggested that I put together a simple app that can be used for the express purpose of testing this approach. The link will take you to a web app version of my calculator, and you can find the source code in my public GitHub repo. If you click this link, the calculator will display in your browser. You can find the source code on GitHub. Additionally, if you're interested, you can find detailed steps on how to deploy this app as a native desktop app for Mac/Linux/Windows in my CalculatorApp_DistributedUsingElectron repo.

Pi Falls from the Universe Yet Again

There is an awesome phenomenon in physics that enables you to calculate digits of pi via elastic collisions. This is inspired by "The Most Unexpected Answer to a Counting Puzzle" on YouTube by 3Blue1Brown, and thoroughly explores the phase space representation of the problem. To get started with this, click my link then when the next page fully loads click the "index.ipynb" file. This will let you experiment with the Python code live.
